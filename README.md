# Tech Challenge I - Diagnóstico Tático Olist
## Análise Exploratória de Fatores Logísticos e Satisfação do Cliente

Este repositório contém o código de desenvolvimento, a documentação e os artefatos analíticos do trabalho da primeira fase da pós-graduação em Data Analytics da FIAP. O objetivo do projeto foi avaliar como a eficiência da operação logística impacta a satisfação do consumidor no ecossistema Olist, identificando gargalos estratégicos por categoria de produto e assimetrias regionais.
